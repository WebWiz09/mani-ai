# MANI — Your AI Pitch Co-Pilot

> Win more clients with MANI. The AI assistant that helps freelancers craft winning proposals, bios, and pitches in seconds.

## What is MANI?

MANI interviews you with a few smart questions, then generates a polished, professional proposal, grant, startup pitch, or bio — tailored specifically to you. No blank pages. No generic templates. Just results.

## Features

- AI-powered interview flow — MANI asks the right questions
- Generates proposals, grants, startup pitches, and professional bios
- Dark and light mode
- Copy, share, and download your output
- Built for freelancers

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Gemini AI (Google)

## Getting Started

1. Clone the repo
   ```bash
   git clone https://github.com/YOURUSERNAME/mani.git
   cd mani
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Add your Gemini API key — create a `.env` file:
   ```
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. Run locally
   ```bash
   npm run dev
   ```

## Deployment

Deployed on Vercel. Add `VITE_GEMINI_API_KEY` as an environment variable in your Vercel project settings.
